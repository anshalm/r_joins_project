# r_joins_project
Hi! This is a project focused on left joins and join_by using 
the popular package 'dplyr' in R. The project covers joins using 
the nycflights23 dataset as well as self-made datasets of car brands, 
car companies, and car parts — hope you leave some feedback!

## Skills Demonstrated
- dplyr — left_join, join_by, select, tibble
- nycflights23 — flights, airlines, weather, planes, airports datasets
- Self made datasets — tibble construction from scratch with c()
- pacman — p_load for package management
